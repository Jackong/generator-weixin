# generator-wxapp
weixin app generator

## install
```shell
npm i -g generator-wxapp
```

## generate project
```shell
yo wxapp
```

## build (generate `dist` dir for weixin app)
```shell
npm run build
```

## watch for livereload 
```shell
npm run watch
```

## generate page
```shell
yo wxapp:page PAGE_NAME
```

## generate template
```shell
yo wxapp:template TEMPLATE_NAME
```
